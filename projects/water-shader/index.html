Stylized Water Shader

Real-time water rendering with foam and caustics for fantasy environments

Category: Shaders
Tools: HLSL, Unity, Shader Graph
Year: 2023

Project Overview

This stylized water shader was developed for a fantasy adventure game requiring dynamic water surfaces with foam generation and caustic light patterns. The shader balances visual appeal with performance, maintaining 60 FPS on mid-range hardware.

Key Features

  • Real-time wave simulation using sine wave combinations
  • Dynamic foam generation based on wave height and shore proximity
  • Caustic light patterns using scrolling textures
  • Depth-based color blending for realistic water depth
  • Configurable parameters for different water types

Technical Implementation

The shader uses a multi-layered approach combining vertex displacement for waves, fragment shader effects for surface details, and screen-space techniques for depth-based effects.

Wave Calculation (HLSL)

float4 CalculateWaves(float2 uv, float time) {
    float2 wave1 = sin(uv * 10.0 + time * 2.0) * 0.1;
    float2 wave2 = sin(uv * 15.0 + time * 1.5) * 0.05;
    float2 wave3 = sin(uv * 20.0 + time * 3.0) * 0.025;
    
    float waveHeight = wave1.x + wave2.x + wave3.x;
    float2 waveNormal = normalize(float2(wave1.y, wave2.y) + float2(wave3.y, 1.0));
    
    return float4(waveNormal, waveHeight, 1.0);
}
